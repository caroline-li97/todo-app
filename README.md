# To-Do List App

A simple full-stack To-Do List application built with React.js, Node.js, 
Express.js, and SQLite.

## Features
- Add a new task.
- View all tasks.
- Delete a task.

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: SQLite
